ProjectMedusaOS

• Ported from Samsung J4+ J415N (v2), Based on J6+ J610F (v3)

• Big thanks to @nnippon and @Asarre2 for help to creating this port

 

Features

• All Samsung securities disabled

• Fully debloated and deknoxed

• Added ProjectMedusaOS wallpaper pack

• Enabled Samsung Screen Recorder

• Enabled Dolby Atmos Game Profile

• Tweaked build.props for decrease lag

• Enabled Samsung call recorder

• Enabled LTE Only mode and network speed indicator and more...

ProjectMedusaKernel

• Custom kernel project for J4+/J6+

• Always open source

 

Features

• Upstreamed to Linux 3.18.140

• Merged with CodeAurora LA.UM.8.6.r1-05300-89xx.0

• Adreno Idler and Adreno GPU Boost

• MSM CPU Hotplug

• KCAL and K-Lapse

• State Notifier, LCD Notifier and Display State

• CPU Voltage Control

• Kernel Sampage Merging (KSM)

• Power Suspend

• FSYNC On/Off and Dynamic FSYNC

• CRC On/Off Switch

• Boeffla Kernel WakeLock Blocker
